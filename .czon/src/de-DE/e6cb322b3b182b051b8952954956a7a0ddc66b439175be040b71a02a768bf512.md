Heute habe ich bei der Überprüfung meines eigenen Codes so etwas gesehen:

**Gegeben: A und B schließen sich gegenseitig aus**

```java
if(A) {
  something1;
  if(B) {
    something2;
  }
  something3;
}
```

Da muss ich mir wohl eingestehen, dass ich nicht ganz bei Sinnen war.
