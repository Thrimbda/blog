> これは私の2025年の総括でしょうか？

もし人を船に例えるなら、理性はその船の舵と帆であり、欲望は船を前進させるエンジン、あるいは風です。

風は帆のない船を押し進めることはできず、船も舵だけでは前進できません。

2025年5月末、私はある種の症状に陥り、私という船は大海原に静かに停滞し、身動きが取れなくなりました。方向もなければ、風もありませんでした。

これは当時の状況に対する私の描写と思考です。当時は何の結論も得られず、ただ自分に一体何が起きているのかを理解し、自分の内面を分析しようと試みていました。

## 症状

2025年5月末から8月にかけて、私は少し迷子になり、「何もしたくない」という状態に陥りました。この状態の原因は、当時はよく分かりませんでした。後から振り返ると、僅かな手がかりを通じてのみ、この原因を遡及的に構築することができます。言い換えれば、異なる現状が与えられれば、これらの解釈も大きく異なるものになったでしょう。

どうしても遡及的に説明するならば、それはおそらく、長年にわたりチーム内の後方支援役として、インフラの構築・補完・日常的な運用を担うことに慣れていたからかもしれません。ここ数年、私が慣れ親しんだ仕事のモードは「与える」ではなく「受け取る」であり、「興味駆動」ではなく「責任駆動」でした。長期的には精神が少しすり減り、疲労と迷いを感じていました。

### 二つの駆動力：責任 vs 興味

> 精神分析の言葉を少し借りれば、これらは大まかに二つの一般的な主体の姿勢に対応します：一つはより「外部秩序に駆動される」もの、もう一つはより「内的興味によって牽引される」ものです。ただし、ここでは厳密な対応付けはせず、違いを説明するために借用します。

この二つの駆動力の違いを具体的に言うと、責任駆動とは意味が主体の外に存在し、既存の秩序を補完してより完全なものにすることです。一方、興味駆動とは意味が主体自身によって付与され、新しい秩序を創造することです。したがって、最近チームの主な任務が「初期状態」にリセットされた後、それまで秩序を構成していた様々な責任の意味は解消され、結果として、やるべきことはたくさんあるのに、どれもやりたがらない、あるいはゆっくりとしか進まないという状態が生じました。

この二つの駆動力は、バランスよく同時に一人の時間とエネルギーを占めるべきものだと私は考えます。天秤がどちらかに過度に傾くと、ある種のトラブルが生じます。過度な責任化は人を疲弊させ、振り返った時に自分が何に忙殺されていたのか分からなくなり、最悪の場合、意味感覚を失います。過度な興味化は、独りよがりになりやすく、外部世界と隔絶したまま自覚せず、多くの外部からのシグナルを無視して自身の内面世界にのみ集中し、結局は外部世界の秩序との関係を処理しなければならなくなります。現在、遡及的に総括すると、過去の一時期、この二つの駆動力は確かに私の中で不均衡な状態でした。

このように分析すると、問題はすでに理解されたと考えられます。私自身にとって、このような迷走した、原動力に欠ける状態から抜け出すためには、私自身が新たに秩序を設定し、様々な問題に意味を付与する必要があり、そうすることで様々な事柄に再び「軽重緩急」の属性が生まれるでしょう。

#### システム的な思考

一時的にペースを落とすことが必ずしも悪いこととは限りません。思考の成果を促し、秩序の再設定に役立つのであれば良いのです。

この思考はシステム的でなければなりません。そうでなければ、幻想と真の思考を区別することができません。私は思考の過程を次のように描写します：人は彼の現状における様々な有利な条件と不利な条件に基づいて、どのように行動すべきかという結論――行動方針――を導き出します。この行動方針は、あらゆる角度からの検証に耐え、それが合理的で、しっかりと立脚しているかどうかを判断されなければなりません。もし合理的であれば、次は当然の行動が続くべきであり、そうでなければ、不完全で未熟な前の結論は新しい結論に取って代わられるでしょう。この過程は、**絶対的**で、変化しなくなるまで繰り返し行うことができます。

この行動方針は、主体が生命の重みを引き受ける必然的な結果です。欠乏が欲望を生み、欲望が主体を駆り立てて、実在する世界を記号化しようと絶えず試みます。しかし、人は自分自身を「完璧な」記号秩序の中に統合して静的な物質と化すことはできません。なぜなら、世界は静的ではありえず、人は自分の記号化された本質が具体的に何であるかを絶対に言い切ることはできないからです。神が死んでから（あるいは人々がついに神は存在しなかったことに気づいてから）、人はもはや自分の本質を把握することができなくなりました（あるいはかつて存在した本質は幻想であったと言えます）。主体が存在し続けるためには、常に実在する世界が変化していることに気づき、それを受け入れ理解し、そして記号システムに再登録しなければなりません。――それはちょうど、流れる水面の油膜が絶えず色合いを変えながら水面を完全に覆おうとするかのようです。この行動方針は未来を指し示し、その未来が主体を自らへと引き寄せるにつれて、主体は絶えず新しく生まれ変わり、陳腐化することも、死ぬこともありません。

#### 思考を放棄する？

私たちは一つの可能性を考慮する必要があります：**人は思考を放棄することが可能でしょうか？**

まず、完全な放棄は、いかなる行動方針も立てることを放棄することを意味します。しかし、人は生まれた瞬間から、生まれる前の無欲無求の神話的状態に戻ることはできません。生理的から心理的までの様々な欠乏が、人にこれらの欠乏を埋めるために何かをしなければならないと駆り立てます。したがって、完全に思考を放棄することは不可能です。

#### 永遠に思考し続ける？

この思考が永遠に揺れ動く振動状態にある可能性はあるでしょうか？

もちろんそのような可能性はあります。このように以前の思考を絶えず覆す再帰的行為の背後には、主体が最終的に一つの行動方針を定めることを拒否し、一つの未来の存在を認めることを拒否しているということがあります。そうすると、時間は主体にとって無意味なものになります。この循環――それは本質的に循環的な逃避であり、存在者としての責任を回避し、この循環の中で享楽することです。確かに、この享楽は相当な魅力を持っています。そうでなければ、なぜTikTok中毒の人があれほど多く存在するのでしょうか。

#### 私の批判

迷いは恥ずべきことです。なぜなら、俗世間と関わらない人は存在しないからです。もし人が飼いならされていないのであれば、彼は自身の現状に基づいてまず行動を強いられなければならず、そうでなければ餓死してしまいます。

## 振り返っての検討

当時の私はここまで書いて行き詰まりました。本来の次の章は「行動方針」であり、上記のような深い思考を経た後に得られた、それに忠実であることができる目標を描写し、その目標から逆算して常にどのように行動すべきかを記述するはずでした。しかし、私は結局それを書き上げることができませんでした。少なくともその時点ではそのような行動方針を得られず、一つの批判を残した後、この文章は放置されました。その原因は、私が理性を失ったことや、私という船自体の性能に何か問題が生じたことにあるのではなく、風、つまり原動力を失ったことにあると私は考えます。

私は絶対的な思考を通じて自身を固定し、すべての不確実性を終わらせる「完璧な」行動方針を求めようとしました。しかし、これこそが私を「思考の享楽」に陥らせました――絶えず推論し、絶えず覆し、思弁的な振動で現実の選択を置き換えたのです。私は後になって初めて、これ自体が症状の一部であると理解しました：外部の意味の保証が機能しなくなった時、私は内省を通じて完璧な記号秩序を構築し、それを置き換えようと幻想を抱いたのです。これは必然的に失敗します。なぜなら、**思考の終着点は完璧な答えではなく、責任を引き受ける決意**だからです。

私はますます、すべての哲学は実践に向かわなければならないと信じるようになりました。ツァラトゥストラは山を下りなければならず、洞窟を出た者も洞窟に戻らなければなりません――それは啓蒙の物語を繰り返すためではなく、人間世界の摩擦の中でこそ、思想は現実のものとなるからです。
したがって、私が終わらせる必要があったのは、思考そのものではなく、覆すことを快感とし、躊躇うことを安全とするような振動でした：それは私があたかも真理に近づいているかのように見せかけながら、実際には引き受けを回避させていたのです。
私は別の生活姿勢に変えたいと思います：まず段階的な方向性を行動に落とし込み、その行動の結果で方向性を修正する。現実を審判とし、矛盾を前進の推進力とし、私がその場でぐるぐる回るのではなく。

そして今、私はすでに満を持して前進しており、理性のレベルでさらに効率を最適化する必要があるため、以前の文章を拾い上げて完成させようとしています。

## 診断
> 欲望（desire）= 要求（demand）- 必要（need）。それは一種の剰余です。

私はこの問題を次のように考えました：原動力としての風やエンジンが、以前の私の病巣でした。

私は、人の原動力は**航海に必要な補給システム**に似ていると気づきました。私たちは時間、注意力、感情を、淡水や燃料のように投じ、三つのものと交換することを期待します：意味（**海図と星の指標**）、反響（**海岸の灯台やこだま**）、そして感覚（**船体自身の状態**）。この収支が不均衡になると……私という船の**補給循環**は中断します。

私にとって、その本質は循環可能な、ラカン的な意味での**欲望の経済圏**であり、この経済圏は以前**破綻**していました。

人の原動力は多くの場合、ある「事実」によって駆動されるのではなく、一連の「言説」によって駆動されます。一つの言葉が別の言葉を引き出し、一つの解釈が別の解釈を引き出します：成功が承認を引き出し、承認が価値を引き出し、価値が安心感を引き出します。意味は鎖のように後ろへ後ろへと繋がり、私たちは自分が一つの目標を追っていると思いながら、実際にはこの鎖の上を絶えず後ろへ歩いています。鎖は生活を理解可能にしますが、それが「一時的に停止する」場所を失うと、人は一種の解釈過剰状態に入ります：考えれば考えるほど、覆すのが速くなり、行動するのが難しくなります。

しかし、もっと隠れたものがあります：この鎖を絶えず動かし続けるのは、多くの場合、目標そのものではなく、目標の背後にあるほんの少しの**名状しがたい誘因**なのです――認められた一瞥、安定した位置、「あと少しで手に入る」という欠如。人はプロジェクトを変え、都市を変え、関係を変えることができます。表面的に追い求めるものは変わりますが、そのほんの少しの誘因はしばしば繰り返し現れます。それを理解することは自責のためではなく、見極めるためです：私は一体何に牽引されているのか、どこで過剰に消耗しているのか、どこで自分のフィードバック源を再配置する必要があるのかを。

例えば、私はかつて株式で一年分の給料を稼いだ友人と話したことがあります。私は、あなたは本当にお金が好きなわけではないと思うと言いました。なぜなら、もしそうならあなたは辞職するか、少なくとも株式にもっと気を配るはずだからです。しかし彼は依然として懸命に働いていました。
明らかに彼にとって、彼の意味の連鎖は以下のようではありません：
```
株式投資 -> 富 -> 安心感 -> 自由 -> 満足
```
仔細に分析すると、ここには多くの考察すべき点があります：一方で、株式投資でお金を稼ぐことは、株式投資が常に富を生み出すこと、あるいは一生食べていけるほどの富を保証できることを意味しません。他方で、お金がもたらす自由が必ずしも満足を意味するわけではなく、彼にとってはむしろ他の人に認められること、社会に承認されることが満足をもたらす可能性が高いのです。ここで私は彼を精神分析するつもりはないので、この辺でやめておきます。

この説明的な例を踏まえて、今、私は診断することができます：以前の私にとって、意味の連鎖は以下の通りでした：
```
後方支援業務 -> システムの安定 -> チームの円滑な運営 -> 承認されること -> 価値感
```
この鎖は、かつて私という船の**見えない錨鎖**であり、一方の端は私の仕事に結びつき、もう一方の端は外部の承認という港に投げ出されていました。主な任務がリセットされると、港の固定点は消え、錨鎖はガラガラと音を立てて虚無に沈み、船も当惑してぐるぐる回り始めました。

そして、この鎖自体が取り囲んでいた大きな目標がゼロに戻った時、この鎖の中間の二つの環は保証を失い、責任の意味は解消され、やってもやらなくても違いがなくなり、やるべきことは依然としてたくさんありますが、どれも脇道のように感じられました。私は惰性でまだ貢献すべきですが、見返りは極めて不確実になり、したがって投資を引き上げる傾向にあり、結果として経済圏は停滞しました。

## 再構築

現実世界の経済圏の救済と同様に、時には債務再編が再起の良い機会となります。私の頭の中の「外部が必ず私の意味を保証してくれる」という信念が崩壊した時、苦痛は避けられませんでしたが、苦痛に続いて訪れたのは変化の機会であり、自らの欲望の経済圏を真に主導し、自らの意味の連鎖を自ら鋳造する可能性であり、それによって政治的に成熟した人間となることでした。

カルヴィーノのこの言葉のように：
