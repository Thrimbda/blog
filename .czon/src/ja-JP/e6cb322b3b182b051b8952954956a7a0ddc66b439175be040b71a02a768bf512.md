今日、自分のコードをレビューしていて、なんと以下のようなものを見つけました：

**AとBは排他的であることが既知**

```java
if(A) {
  something1;
  if(B) {
    something2;
  }
  something3;
}
```

これは、私の頭がどこかおかしくなっていたに違いないと、本当に思います。
