Hoy, al revisar mi propio código, me encontré con esto:

**Se sabe que A y B son mutuamente excluyentes**

```java
if(A) {
  something1;
  if(B) {
    something2;
  }
  something3;
}
```

Entonces realmente debo tener algún problema en la cabeza.
