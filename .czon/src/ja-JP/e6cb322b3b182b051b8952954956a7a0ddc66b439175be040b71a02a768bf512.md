今日、自分のコードをレビューしていて、こんなものを見つけました：

**AとBは排他的であることが既知**

```java
if(A) {
  something1;
  if(B) {
    something2;
  }
  something3;
}
```

これを見て、私は本当に自分の頭がどうかしてしまったんじゃないかと思いました。
