来记录我的一天！ 今天换种格式来记录

1.  chatgpt pro

    我周五的时候心一横，去万能的闲鱼上买了一个 chatgpt pro，要价 1369，我一看这比 200USD 要便宜，就爽快下单。结果到周二才给我弄好，他给了我一个outlook 邮箱和一个绑定了这个邮箱的 chatgpt 账号。

    我登录了 outlook 邮箱一看，好家伙，有一个 chatgpt 账单，菲律宾比索付款的。

    |                          |                 |
    | ------------------------ | --------------- |
    | plan                     | amount          |
    | ChatGPT Pro Subscription | ₱8919.64        |
    |                          | Tax: ₱1070.36   |
    |                          | Total: ₱9990.00 |
    | Payment method           | Mastercard-xxxx |

    我好奇就去查了一下折合 cny 的价格，竟然只要 1174.54 CNY 左右。等于他净赚我将近 200 块钱！

    然后我去调研了一下，奶奶滴。这钱该他赚啊，菲律宾确实是全球价格最低的能买到 chatgpt pro plan 的地方。牛逼。

    我有可靠的线人在泰国，我想复制他的成功，结果哈哈泰国更贵。折合 cny 1537，那我有没有可能去比如波兰（1942 CNY）的类似闲鱼的地方卖菲律宾的 chatgpt会员呢？🤔

2.  http-proxy 的了结

    终于把 http-proxy 的事情搞通了，我滴妈呀，比我想的要复杂，这也体现了legion 在一个 mono-repo 内对同一个 topic 横跨 8 个 project 的工作能力几乎就是极限了，我甚至碰到有几次 subagent 闪退的问题。

    但是话又说回来了，没有 legionmind 的时候，我印象直接给 ai 下达指令是根本搞不定这种程度的问题的。这也间接说明了 legionmind 在这种时候体现的独特能力。

    最后这个任务还是被 Legion 自己搞定了，我现在基本可以脱离编码，甚至可以在大多数情况下仅留下少量的针对设计文档的 review comment 了。我对这个版本的legion 已经比较满意了。

3.  Legionmind 的迭代

    说到 legionmind。

    我今晚又升级了一下 legion，我觉得我给他规定死一个 workflow 还是有点僵化，本质上还是在我的框架内去让 AI work。但是 AI，尤其聪明的 AI：比如我开了chatgpt pro 之后常开的 codex 5.2 extra high，其实他什么都懂。我其实也不知道是否我这套 multi-agent 系统是不是在这方面有所浪费。我觉得随着 AI 越来越聪明，让它自己设计流程应该是比较合理的工作方式吧。在这个层面上，我也好，manus 也好，oh-my-opencode 也好，本质上都是在这个 AI 可以完全取代人类去做现有的人类工作之前的铺路人。就如同尼采说的，人是人自己通向超人的路啊。

    那么言归正传，应该以科学的方式，系统地测试不同版本的 legion 的表现，应该有一套 benchmark 来测试不同的 legion 在面对同样的一组复杂编码任务的表现。我应该尽快搞出这样一种 benchmark 工具（也许可以参考现在业界常见的 bench？）

    接下来除了继续 Yuan 的工作以外，就要在这个方面去努力。

    说起来之前那个 legionmind github bridge 我这几天没再继续搞了，原因是我看到 opencode 其实自己就有这个能力。与其硬着头皮做完还满是 bug，不如暂时先hold。我认为还是暂时不要做容易被海浪拍死的事情，要做一艘随着海面上升一起上升的船。（这个比喻抄自 manus 的博客）

    —今天就这么的吧，CZ 的跨服评论我是看到了。我觉得确实该想记的时候记一笔，不过习惯是要去花时间养成的，记录与我而言最大的意义不是以后的某一天来回顾这一天的细节，而是记录的这个动作本身就是我在进行思维链式的深度思考的过程，即使这个日志没有任何人看，我的记录对我自己而言也依然有相当大意义。
