## Diseño Detallado

### Flujo Principal
1.  Modificar la lógica de salida de imágenes en `get-gcores-talk.ts`: Cambiar la macro `slideshow` por múltiples líneas independientes de imágenes markdown, cada una con `![...](...)`.
2.  Modificar `templates/shortcodes/slideshow.html`: Generar una estructura HTML compatible con Embla (`.embla`, `.embla__container`, `.embla__slide`), manteniendo el contenedor externo `.slider-container` para preservar la compatibilidad de estilos.
3.  Agregar etiquetas de script CDN para Embla y Embla Autoplay en `templates/head.html`.
4.  Agregar código de inicialización de Embla en `static/js/script.js` (o crear un nuevo script), para que inicialice automáticamente los contenedores `.embla` y omita los que ya estén inicializados.
5.  Agregar los estilos CSS de Embla de CZON a `static/css/style.css`, o incrustarlos mediante una etiqueta style.
6.  Eliminar el código JavaScript del slider existente en `static/js/script.js` (o conservarlo pero deshabilitarlo).

### Definición de Interfaz
- Estructura del contenedor Embla:
```html
<div class="slider-container">
  <div class="embla">
    <div class="embla__container">
      <div class="embla__slide"><img src="..." /></div>
      ...
    </div>
  </div>
</div>
```

### Detalle de Cambios en Archivos
- `scripts/get-gcores-talk.ts`: Modificar la lógica de salida de imágenes.
- `templates/shortcodes/slideshow.html`: Reescribir para usar la estructura Embla.
- `templates/head.html`: Agregar etiquetas de script de Embla.
- `static/js/script.js`: Agregar función de inicialización de Embla, eliminar código del slider existente.
- `static/css/style.css`: Agregar estilos de Embla, ajustar estilos del slider existente.

### Puntos de Incertidumbre
- ¿Es necesario reescribir completamente los estilos CSS del slider existente? Se pueden conservar los estilos del contenedor externo `.slider-container`, y usar nuevos estilos para la estructura interna de Embla.
- ¿La lógica de agrupación automática de Embla podría afectar inesperadamente a imágenes en otras páginas? Es necesario asegurarse de que la inicialización solo se aplique a grupos de imágenes que contengan la clase `.embla` o contenedores específicos.
