Hoy, al revisar mi propio código, me encontré con algo así:

**Se sabe que A y B son mutuamente excluyentes**

```java
if(A) {
  something1;
  if(B) {
    something2;
  }
  something3;
}
```

Realmente creo que mi cerebro debió tener algún problema en ese momento.
