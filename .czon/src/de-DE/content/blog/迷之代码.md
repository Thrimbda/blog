**Gegeben: A und B schließen sich gegenseitig aus**

```java
if(A) {
  something1;
  if(B) {
    something2;
  }
  something3;
}
```

Da dachte ich wirklich, dass mit meinem Kopf wohl etwas nicht stimmt.
