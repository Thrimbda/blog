Hoy, al revisar mi propio código, me encontré con esto:

**Dado que A y B son mutuamente excluyentes**

```java
if(A) {
  something1;
  if(B) {
    something2;
  }
  something3;
}
```

Realmente debo haber tenido algún problema mental en ese momento.
