Heute habe ich bei der Überprüfung meines eigenen Codes so etwas entdeckt:

**Es ist bekannt, dass A und B sich gegenseitig ausschließen**

```java
if(A) {
  something1;
  if(B) {
    something2;
  }
  something3;
}
```

Da muss ich mir wohl ernsthafte Sorgen um meinen Verstand machen.
